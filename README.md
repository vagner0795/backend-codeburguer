# Api CodeBurguer


### Resumo

O projeto trata-se de um API com funções:
 - Login
 - Cadastro de usuário
 - Cadastro de novos produto
 - Cadastro de categoria dos produtos
 - Atualização de Produtos, Usuários e Categorias
